## 🧪 Main Features

### EWIS Suite
EWIS helps test how an AI system handles unsafe or unwanted input. It can expose weak spots in prompt handling and policy checks.

### WOIK Suite
WOIK helps you run broader workflow checks across multi-step agent setups. Use it to see how a system behaves when tasks depend on several actions.

### PHIAK Suite
PHIAK focuses on risk checks that matter for higher-stakes systems. It helps you review safety paths and decision limits.

### Local and Air-Gapped Use
ARES-E works well in closed environments. You can keep your test work separate from outside networks.

### Continuous Red-Teaming
You can run checks again and again as your AI system changes. That helps you track whether a fix still works after updates.

### Report Output
ARES-E can produce clear test results so you can review what happened and share findings with your team.
